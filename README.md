# Docker Concourse Util

A general-purpose Docker image that provides dependencies for 
common [Concourse CI](https://concourse-ci.org) tasks.

Includes:
* `curl`
* `docker`
* `fish`
* `git`
* `gitflow`
* `jq`
* `make`

## Usage

```bash
docker pull troykinsella/concourse-util
```
